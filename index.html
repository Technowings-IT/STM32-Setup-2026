STM32F746G-DISCO — Getting Started

This is a hands-on, fully offline guide that takes someone new to embedded systems from zero to four working projects on the STM32F746G-DISCO board. Every page works by double-clicking this file — no internet connection needed.

The software stack

ToolVersionWhat it's for
STM32CubeIDE2.1.0Where you write, build, flash, and debug your code (bundles the CubeMX engine)
STM32CubeMX6.17.0Visually configure the chip (pins, clocks, peripherals) and generate C code
TouchGFX4.26.1Drag-and-drop designer that generates the C++ for on-screen graphics
HALHAL1 (classic)ST's ready-made C functions the generated code calls
Note A newer CubeMX2 / HAL2 line exists, but it is less mature — beginners should stay on classic CubeMX 6.x + HAL1, which this guide uses.

Recommended path

  1. Read pages 01 (Software Stack), 02 (Board Overview), and 03 (Install & Setup) first to get your tools installed and understand the hardware.
  2. Then work through the tutorials in order: 04 Blink LED → 05 Button + UART → 06 TouchGFX Hello World → 07 Interactive UI. Each tutorial builds on the one before it, so don't skip ahead.
  3. Keep 08 Troubleshooting & Glossary open in a tab as a reference whenever you hit an error or an unfamiliar term.
