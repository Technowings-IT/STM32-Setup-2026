Install & setup (Windows)

This page walks you through every tool you need before writing your first line of STM32 code. All downloads come from st.com, are free, and — since CubeMX 6.16 — no ST account is required.

What to install

  1. STM32CubeIDE 2.1.0 — the main IDE you will use for editing, building, and flashing code.
  2. TouchGFX 4.26.1 — the graphical UI framework for the STM32F746G-DISCO display. You can either install the standalone TouchGFX Designer application, or add it from inside CubeMX via Help → Manage embedded software packages → Application templates.
  3. STM32CubeMX 6.17.0 — the graphical configuration tool used for pin configuration, clock setup, peripheral configuration, and middleware integration. In STM32CubeIDE 2.x, CubeMX is installed and used as a separate application.
Note Downloads are free and account-free since CubeMX 6.16.

Install order & steps

  1. Go to st.com and download the STM32CubeIDE 2.1.0 installer for Windows.
  2. Run the installer. Accept the license agreement and click through the setup wizard.
  3. When Windows or the installer prompts you to install the ST-LINK USB driver, accept it. This driver lets your PC talk to the board.
  4. Finish the CubeIDE installation.
  5. Download and install STM32CubeMX 6.17.0 the same way — run its installer and follow the prompts. Do the same for TouchGFX Designer 4.26.1.

First launch

Open STM32CubeIDE from the Start menu. On first launch it asks you to choose a workspace folder — this is simply the folder where your projects will be stored. You can accept the suggested path or pick your own. Click Launch. You will land on the Information Center welcome screen.

Connect the board

Plug a USB cable from your PC into the ST-LINK USB port on the STM32F746G-DISCO board (labelled USB ST-LINK). A power LED on the board lights up. Windows will automatically install the ST-LINK driver (if it did not install during setup) and create a virtual COM port — a simulated serial port that lets you send debug output over USB.

Watch out Use a USB data cable, not a charge-only cable. Charge-only cables carry power but no data, so the board will appear to power on but your PC will not detect it.

Verify the connection

To confirm everything is recognised by Windows:

  1. Right-click the Start button and choose Device Manager.
  2. Expand Ports (COM & LPT).
  3. You should see an entry such as STMicroelectronics STLink Virtual COM Port (COM3). The exact number (COM3, COM4, …) varies per machine — make a note of it, you will need it in Tutorial 2.
  4. Optionally, if STM32CubeIDE later prompts you to upgrade the ST-LINK firmware, accept the upgrade (you can also do this at any time through STM32CubeProgrammer).
✅ Checkpoint Tools installed, board powered, and the ST-LINK together with its virtual COM port appear in Device Manager.
Watch out If no COM port appears or you see "ST-LINK not found", see Troubleshooting.
